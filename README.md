
# Software-Based Energy Efficiency Auditing System

**Author:** Jerry Kennedy  
**Institution:** Adventist University of Central Africa (AUCA), Kigali, Rwanda  
**Department:** Software Engineering  
**Partner Organization:** Energy Utility Corporation Limited (EUCL), Rwanda  
**Project Status:** Undergraduate project – approved, conception/design phase

## Project Overview
This project proposes a software-based system to automate energy efficiency auditing processes. The system is designed to support EUCL’s mission to improve energy efficiency, reduce energy losses, and enhance data-driven decision-making through accurate analytics and automated reporting.

## Objectives
- Automate energy consumption data collection and auditing
- Analyze energy efficiency using statistical and analytical models
- Generate digital audit reports
- Support predictive analysis to identify inefficiencies
- Ensure secure, role-based access to sensitive data

## Scope
This repository contains the official project concept and requirements document submitted as part of the AUCA undergraduate Software Engineering program. The implementation and evaluation phases are forthcoming.

## Verification
This repository serves as a public academic artifact verifying the existence and scope of the approved undergraduate project.
